### Core plugins
- **Tag pane** List the tags and their number of occurrences.
- **Outline** Show the outline of a note.
- **Daily notes** to create journal notes.
### Community plugins
These are some useful plugins you can install on your obsidian. Some of them are mandatory if you want to use this skeleton (day planner, templater, dataview).
- **Advanced Tables** To be able to create tables easily.
- **Dataview** your notes will be treated as a database so you'll be able to query them.
- **Day Planner** Your daily notes more accesible.
- **Emoji Toolbar** To use emojis on your notes.
- **Kanban** To keep a record of your ongoing and to do stuff.
- **Tasks** Track tasks across your entire vault.
- **Templater** Automatize the creation of some types of notes.
- **Excalidraw**. Draw on excalidraw inside Obsidian.
- **Editor syntax highlight** for code highlight in editor mode.
- **Emoji toolbar** for looking up emoji (don't forget to customize its shortcut, currently `Cmd + ;`).
- **Tag wrangler** for easy renaming and merging of tags.
- **Calendar** for a fancy calendar pane.
